# KOSPI Stock PWA

KOSPI/국내 주식 분석용 PWA 예제입니다.

## 기능
- KOSPI / 국내 종목 조회
- SMA20 / SMA60 / RSI14 / MACD
- 5일 단순 예측
- 관심종목 저장(LocalStorage)
- 다크모드
- PWA 설치 지원
- GitHub / Vercel 배포 가능

## 실행
```bash
npm install
npm run dev
```

## 배포
- GitHub에 push
- Vercel에서 Import 후 Deploy

- 005930.KS
- 000660.KS
